ABOUT MINERVA

Minerva is the application built to provide a more interactive experience with MKProject Books, and this is done by giving a more one-stop place to read the various books in both web & pdf formats. The version scheme of this application follows the M.m.p format, where:

  • M: Major (Major Improvements to the application)
  • m: Minor (Adding new books to the library)
  • p: Patches (Fixing weird bugs, spelling mistakes, etc.)

I hope to make this application as good as it can be, and hope to get support to make this dream even more possible.
